CSV Format Guide

Learn how to format your CSV file correctly

Your CSV file should have voucher codes in the first column. Here's a perfect example:

# WiFi Voucher Codes
# Lines starting with # are ignored
voucher_code
VOUCHER001
VOUCHER002
VOUCHER003
# Guest access codes
ABC123DEF
XYZ789GHI
Pro Tips:
  • • Headers are completely optional
  • • Lines starting with # are treated as comments and ignored
  • • This allows you to organize your voucher codes with descriptions
  • • Empty lines are automatically skipped
